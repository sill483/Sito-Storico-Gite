# Sito-Storico-Gite
Il progetto “Storico Gite”, un sito web pensato per migliorare la gestione delle gite scolastiche all’interno dell’istituto Ettore Majorana.
Progetto rimasto incompleto in alcune sue parti, in particolare, interfaccia inserimento e tasto visualizzazione gite.

Il sistema consente la gestione strutturata di gite, classi, docenti accompagnatori e coordinatori, con ruoli differenziati e accessi controllati. Composto da:

Fase di progettazione 
-Modellazione del database tramite diagramma E‑R.
-Definizione delle regole aziendali, tra cui:
una classe può effettuare una sola gita all’anno
un docente può accompagnare più classi ma in date diverse
i coordinatori possono modificare solo le gite da loro inserite
-Stesura delle specifiche funzionali e dei vincoli del sistema.

Sviluppo front-end -Progettazione e realizzazione dell’interfaccia grafica del sito.
-Utilizzo di HTML, CSS e JavaScript per la struttura e l’interattività.
-Integrazione del framework Bootstrap per garantire un layout responsive e moderno.
-Implementazione delle pagine principali e gestione dei ruoli utente.
-Utilizzo dell’ambiente di sviluppo Visual Studio.

Sviluppo back-end
-Implementazione della logica applicativa tramite PHP, per la gestione dinamica delle pagine e delle funzionalità.
-Creazione e gestione del database tramite SQL.
-Utilizzo di XAMPP e phpMyAdmin come ambiente di sviluppo locale.
-Collegamento tra front‑end e database per rendere il sito completamente dinamico e interattivo.
